## 3. 安全提醒

### 3.1 密钥保管

- **你自行负责**保管所有密钥，包括但不限于：
  - Cloudflare API Token
  - `ENCRYPTION_KEY`（加密密钥，丢失后已加密的 API Key 将无法解密）
  - `JWT_SECRET`
  - `ADMIN_BOOTSTRAP_CODE`
  - 管理员账号密码
  - 统一 API Token（`freellmapi-xxx`）
  - 各上游提供商的 API Key
- 请使用密码管理器保存所有密钥
- **切勿**将密钥提交到 Git 仓库或在公开场合分享
- 如果 `ENCRYPTION_KEY` 丢失，所有已加密的上游 API Key 将永久无法解密

### 3.2 部署安全

- 部署后请立即修改默认的管理员启动码
- 建议启用 Cloudflare 的 WAF 和速率限制功能
- 建议为管理面板配置访问控制（如 IP 白名单）
- 定期检查 Analytics 页面，关注异常请求模式

### 3.3 上游 API Key 安全

- 本项目将你的上游 API Key 加密存储在 D1 数据库中
- 添加密钥后仅在页面上显示一次明文，请立即保存
- 请确保你使用的上游 API Key 符合对应提供商的使用条款
- 请勿使用盗取或非法获取的 API Key
